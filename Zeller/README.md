# Zeller Checkout System

A simple checkout system for Zeller's computer store.

## Steps to Run

### 1. Install Dependencies
```bash
npm install
```

### 2. Run Demo Scenarios
```bash
npm run demo
```

### 3. Run Tests
```bash
npm test
```

### 4. Build Project
```bash
npm run build
```

### 5. Run Built Project
```bash
npm start
```

## Commands Available

```bash
# Run demo scenarios
npm run demo

# Run tests
npm test

# Build project
npm run build

# Run built project
npm start

# Development mode
npm run dev
```

## Example Scenarios

**Scenario 1**: atv, atv, atv, vga → $249.00 (3-for-2 deal)  
**Scenario 2**: atv, ipd, ipd, atv, ipd, ipd, ipd → $2718.95 (bulk discount)

## Pricing Rules

1. **3 for 2 Deal on Apple TVs**: Buy 3, pay for 2
2. **Bulk Discount on Super iPad**: $499.99 each when buying more than 4
